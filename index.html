Elephant Accident Prevention
National parks and wildlife reserves in Western and Northern Indian states have railways which cross prominent elephant corridors. Sometimes elephants are ran over by trains. This is a huge issue and can't be managed by forrest guards due to lack of resources. We are trying to build a low cost early warning system to alert train drivers or station masters when elephants are near railway lines. Cameras are quite expensive when installed on a large scale and are dependent on limited visual information. Proximity sensors give false alarm due to their sensitivity towards small disturbances, which can be caused by any animal, elephant or not. We use seismic sensors as they can detect small seismic activities and a herd of elephants makes enough disturbance to trigger a seismic sensor. The challenge lies in classifying seismic activities as caused by elephants or caused by a train. We are using time series analysis to understand seismic data and classify them. This project is funded by World Wildlife Fund(WWF) and Wildlife Institute of India(WII).

Collaborative Filtering with Label Consistent Restricted Boltzman Machine[Code][Dataset]
The possibility of employing restricted Boltzmann machine (RBM) for collaborative filtering has been known for about a decade. However there has been hardly any work on this topic since. This work revisits the application of RBM in recommender systems. RBM based collaborative filtering only used the rating information; this is an unsupervised architecture. This work adds supervision by exploiting user demographic information and item metadata. A network is learnt from the representation layer to the labels (metadata). The proposed label consistent RBM formulation improves significantly on the existing RBM based approach and yield results at par with the state-of-the-art latent factor based models.

Real Time Image Segmentation in Drone Imagery
Developed a real time image segmentation for DJI phantom quadrotor. UAV is able to do scence segmentation on three different types of scence, when UAV is inside building, outside building with camera plane perpendicular to UAV and when camera plane is parallel to UAV. Pretrained SegNet model on CamVid road dataset and SUN RGB-D dataset have been used for first two scenarios, respectively. For third scenario network is trained from scratch on images taken by DJI phantom. For the same manual annoation has been done. Dataset and codes will soon be released.

Floating Sensor Network[Code]
We are implementing a system to monitor the pollution level in waters of Indian rivers. Current method involves collecting pollution data at fixed intervals and only at fixed locations. We have designed a floating sensor unit that consists of pH meter, temperature sensor, turbidity meter, conductivity meter and ORP(Oxidation-Reduction Potential) meter to collect pollution related information. We use a network of such sensors and collect their data with geolocation using GPS and transmit it via SMS/Data pack to a central server. The collected data will be used to continously monitor pollution level and draw useful information about the pollutants and the location where they enter the river. We will also release a heat map showing each sensor's value on a map.

Multi Modal Search[Code]
We focus on the problem of searching when user has limited information about the topic but can represent the query through different modes, such as image, text and speech. Our system uses Convolutional Neural Network to understand an image and give a description along with the class to which the image belongs. For the text part, we use Long Short Term Memory network to expand a query. If speech is present, we simply convert it to text and apply the LSTM part to it. The query can comprise of any combination of three modes. We merge the output of LSTM and CNN to enrich the querry and then search it in the elasticsearch database to get relevant information. Our dataset contains images, speech and text linked together from wikipedia.

UAV Scheduling Algorithm for Multiple Target Tracking[Code]
We focus on the problem of tagging multiple vehicles using multiple UAVs in a resource constrained environment. We want to minimize the total tagging time and the number of UAVs. Ideal solution involves creating dynamic graph and running a greedy travelling salesman problem. For real world scenario, we consider arrival rate and do graph partitioning based on the arrival rates to fairly allocate tagging task among multiple UAVs.

Scholar's OWL[Code and Dataset]
We have developed a semantic search system for searching scholarly articles. Current methods involve full text search and have high recall but low precision. Existing search engines like google scholar, IEEE explorer, researchgate don't use semantic information like content of paper, authors, publishers, topic, sub topic, etc. We believe that when we link papers according to topic, subtopic, authors and other relevant information we can search more accurately. For this we use the techniques of semantic web and represent articles and related information in Web Ontology Language(OWL). Then we use SPARQL to query the OWL graph. We have also used elasticsearch to implement full text searching. We create a heirarchical structure of topics, subtopics and papers so that users can easily navigate to related papers under similar topics. For this we have used heirarchical clustering and applied topic modelling over the clusters obtained.

State Estimation using SLAM in Swarm Robotics
We have solved the problem of efficient map construction of an indoor environment using swarm robots. For this we have used SLAM and triangulation method to know locations of each bot with respect to every other bot. We have used Kalman Filter to remove uncertainities in the state calulation.
